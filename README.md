# 🎨 Logo & Doodle Designer

Create your own **logos, sketches, and doodles** using Python & Streamlit!  

---

## Features
- Free-hand drawing with brush size & color picker  
- Clear canvas anytime  
- Save your creations as images  
- Simple, beginner-friendly UI  
- Works without external components (only Pillow required)  

---

## How to Run

```bash
pip install streamlit pillow
streamlit run drawing_app.py
